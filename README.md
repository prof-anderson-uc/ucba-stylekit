# 📦 UCBA Stylekit CSS

A beginner-friendly CSS stylesheet designed for **UC Web Design students** to help you quickly style your web pages while learning essential layout and design concepts.

---

## 🧰 What is the Stylekit?

The **Stylekit** is like a toolbox or collection of **premade CSS classes** that you can add to your HTML elements. It helps you do common layout and design tasks without having to write the CSS from scratch.

It includes styles for:

- Typography
- Buttons
- Layout (Flexbox/Grid)
- Spacing & utility
- Colors
- Borders & shadows
- Images
- Transitions

---

## 💡 Why Use This Stylekit?

This stylekit allows you to:

- Quickly apply styles without writing new CSS
- See how changes in CSS affect your HTML
- Learn the logic and structure behind **CSS frameworks**
- Focus on content and layout while still having clean design

---

## 📚 How It’s Like a Framework (But Simpler)

Popular CSS frameworks like **Bootstrap** and **Tailwind** work the same way — they offer **predefined class names** that you apply to your HTML to make it look and behave a certain way.

This stylekit is **a simplified version**, designed to teach you how:

- Styles can be separated into classes
- You can build layouts using combinations of class names
- Class naming follows clear, consistent rules

---

## 🔧 What is a Framework and Why Are They Popular?

A **CSS framework** is a pre-written collection of CSS (and often JavaScript) that:

- Speeds up development
- Ensures consistency across designs
- Helps with responsiveness and accessibility

Frameworks like Bootstrap, Tailwind, and Foundation are widely used in industry to **save time** and **maintain quality** in large websites.

---

## 🏷️ Naming Conventions

Class names are short, readable, and consistent.

- **Typography:** `.text-lg`, `.bold`, `.italic`
- **Spacing:** `.m-1` (margin), `.p-1` (padding)
- **Layout:** `.flex`, `.grid`, `.center`
- **Buttons:** `.btn`, `.btn-primary`
- **Coloring:** `.bg-primary`, `.text-muted`
- **Utilities:** `.divider`, `.shadow`, `.rounded`

This helps you remember, reuse, and recognize patterns easily.

---

## 🚀 How to Use It

1. Link the stylesheet in your `<head>`:
   ```html
   <link rel="stylesheet" href="ucba-stylekit.css">
   ```

2. Use the classes in your HTML:
   ```html
   <section class="card text-center p-1 bg-light">
     Welcome to my website!
   </div>
   ```

3. Preview the demo HTML (`stylekit-demo.html`) to see examples in action.

---

## ✅ When Should I Use the Toolbox?

- Building quick layouts
- Trying different text sizes, alignments, and effects
- Learning how class-based CSS works
- Making your page more visually appealing without deep custom CSS

---

## 📌 Things to Remember

- This is **not a replacement for writing your own CSS**, it’s a learning tool.
- Use class-based styling here to learn, then build your own styles later.
- This is **pure CSS only** — no JavaScript or interactivity is included.
- All styles are optional. Use what makes sense for your project.
